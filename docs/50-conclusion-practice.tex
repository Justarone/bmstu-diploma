\section*{ЗАКЛЮЧЕНИЕ}
\addcontentsline{toc}{section}{ЗАКЛЮЧЕНИЕ}

В рамках данной работы:
\begin{itemize}
	\item [---] были описаны инструменты и технологии, требуемые для реализации нового метода;
	\item [---] был реализован новый метод в виде класса, отвечающего за управление цепью хеш-сумм, связывающую куски данных;
	\item [---] полученная реализация класса была встроена в функционал движка MergeTree;
	\item [---] метод был исследован на предмет возможности реализации угроз (обхода защиты) при нешифрованном хранении данных;
	\item [---] была исследована корректность работы метода при различных событиях системы при шифрованном хранении данных.

Таким образом цель работы --- реализовать улучшение метода блочного хранения данных в СУБД ClickHouse в движке MergeTree, дающее возможность доказательства неправомерного доступа была достигнута.
